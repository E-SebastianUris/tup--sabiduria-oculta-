# Curriculum Web - Melany Besguro

Este proyecto fue desarrollado como trabajo práctico para la materia Desarrollo Web.
Consiste en un currículum vitae digital construido con HTML, CSS y JavaScript.

## Funcionalidades
- Diseño estructurado con HTML5
- Estilos personalizados con CSS3 y variables
- Tarjetas interactivas de habilidades
- Botón de navegación para volver al inicio
- Formulario de contacto

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- GitHub Pages
